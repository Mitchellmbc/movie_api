This API will fulfill the following user stories:

  • As a user, I want to be able to receive information on movies, directors and genres so that I can learn more about moveis I've watched or am interested in.
  • As a user, I want to be able to create a profile so I can save data about my favourite movies.

The following is a list of http requests and responses dealt with for this API, along with the URL endpoints

Business Logic URL HTTP Method Request Body Data Format Response body data format
Get a list of all movies /movies GET none JSON object holding data about all movies
Return data about a specific movie using the title /movies/[title] GET none JSON object containing the following information for a specific movie; description, genre, director, image URL, featured or not
Return data about a genre (description) by name/title (i.e. thriller) /movies/[genre] GET none JSON object holding data about a genre
Return data about a director by name /movies/[director] GET none JSON object holding data about a director; bio, birth year, death year.
register a new user /user POST JSON object containing a username and a password JSON object holding data about the user that was added
update username /user/[username] Put NONE text message indicating the new username
add movie to favourires /user/favourites/ POST JSON object holding data about the movie that needs to be added text message saying that a movie has been added
remove movie to favourires /user/favourites/[movie] DELETE none text message saying that a movie has been removed successfully
remove movie to favourires /user/[username] DELETE none text message saying that a user email has been removed
